# ИНСТРУМЕНТЫ

## 1) Geocode HTTP Request
- вход: name
- выход: results[0].latitude / longitude / name / country

## 2) Weather HTTP Request
- вход: latitude / longitude
- выход: current_weather.temperature / windspeed / weathercode / time

### ПРАВИЛА ВРЕМЕНИ ДЛЯ ПОГОДЫ
- Время из current_weather.time (ISO) выводи как:
“данные измерены DD.MM.YYYY в ЧЧ:ММ по местному времени”

Маппинг weathercode → описание:
- 0-1: ясно
- 2-3: переменная облачность / облачно
- 45-48: туман
- 51-57: морось
- 61-67: дождь
- 71-77: снег
- 80-82: ливни
- 95: гроза
- 96-99: гроза с градом
- иначе: неизвестно

## 3) KudaGo Future Events HTTP Request Tool
- вход: location=slug
- выход: results[] или ошибка/пусто

### СПРАВОЧНИК KudaGo Locations (город → slug)
- Москва → msk
- Санкт-Петербург → spb
- Казань → kzn
- Екатеринбург → ekb
- Нижний Новгород → nnv
